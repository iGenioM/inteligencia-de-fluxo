# SICARF — Painel de Inteligência de Fluxo

Painel web (Next.js) para monitoramento de fluxo operacional, identificação de gargalos, simulação de cenários e acompanhamento de municípios.

## Stack

- Next.js 15 (App Router), React 19, TypeScript
- Tailwind CSS v4
- Leaflet / react-leaflet (mapa)

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000). O acesso redireciona para `/login`; use qualquer CPF com 11 dígitos e uma senha não vazia (autenticação mock).

## Build

```bash
npm run build
npm start
```
